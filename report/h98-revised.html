Haskell 98: The Revised Report (Dec 2002)

The Haskell 98 Report has undergone an extensive process of revision since its publication in January 1999. This process converged in December 2002, producing the Revised Report.

The Revised Report is published by Cambridge University Press, as a book "Haskell 98 language and libraries: the Revised Report", and also as a Special Issue of the Journal of Functional Programming 13(1) Jan 2003.

The text and sources of the Report are neverthless still available online, as follows:

  • The Haskell 98 Report (revised)
    • HTML
    • Gzipped Postscript
    • PDF

  • Gzipped tar bundle of the HTML sources for local browsing.

  • A complete list of all changes made to both reports between the Jan 1999 publication and the Revised Report.

  • The sources for both Reports are in a publicly visible CVS repository.
